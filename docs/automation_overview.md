# Automation Overview

This project contains multiple automation scripts to help simplify daily tasks.
The goal is to provide examples of Python, Bash, and Docker-based automation.

## Included Scripts

### 1. sample_hello.py
A basic Python script used for demonstration.

### 2. backup_folder.py
Creates a backup of a specified directory and stores it in a zip/tar file.

### 3. cleanup_temp.py
Removes temporary files from the `temp/` folder.

### 4. system_info.sh
A Bash script that prints basic system information such as:
- OS version  
- CPU info  
- Memory and storage usage  

## CI/CD Integration
The project uses:
- GitHub Actions for continuous integration  
- Docker for isolated runtime  
- Shell scripts for formatting, linting, and testing  

Use this documentation as a quick reference on how automation is structured.
